## 一、本周学习概览
主要依照javaSE的课程进行学习，从自己没有学过的知识点开始看网课，主要学习内容如下
- **集合框架**：学习了 ArrayList、LinkedList、HashMap 的底层结构，重点看了 HashMap 的扩容机制和哈希冲突解决。

- **Stream流**：学会了 filter、map、collect 等常用操作，能用流式处理替代传统的 for 循环。

- **IO流**：掌握了字节流/字符流的区别，学会了用 BufferedReader 高效读取文件，以及 Properties 配置文件的使用。

- **日志技术**：配置了 Logback，了解了日志级别（TRACE/DEBUG/INFO/WARN/ERROR），不再用 System.out.println。

- **多线程**：学习了线程创建的3种方式，理解了 synchronized 和 Lock 的区别，以及线程池的核心参数含义

## 二、遇到的问题与解决
**问题** IO类太多，分不清什么时候应该用那个
**解决** 根据名称进行分类，（比如：以 Input 结尾 / 带 Reader → 读，以 Output 结尾 / 带 Writer → 写），并加以练习。
